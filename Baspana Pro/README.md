# Baspana Pro — Веб-сайт строительной компании

## Структура проекта

```
Baspana Pro/
├── index.html          # Главная страница
├── css/
│   ├── style.css       # Основные стили
│   └── animations.css  # CSS-анимации
├── js/
│   └── main.js         # JavaScript (интерактивность)
└── images/
    ├── logo.png        # Ваш логотип (добавьте сюда!)
    └── logo.svg        # Резервный SVG-логотип
```

## Быстрый старт

Просто откройте `index.html` в браузере — сайт работает без сервера.

## Добавление вашего логотипа

Скопируйте ваш файл логотипа в папку `images/` и назовите его `logo.png`.
Сайт автоматически подхватит его.

## Разделы сайта

1. **Hero** — Главный экран с фоном и призывом к действию
2. **Marquee** — Бегущая строка с услугами
3. **О компании** — История и преимущества
4. **Услуги** — 6 карточек услуг
5. **Этапы работы** — 4-шаговый процесс
6. **Портфолио** — Галерея с фильтрацией
7. **Преимущества** — Почему выбирают нас
8. **Отзывы** — Слайдер с 5 отзывами
9. **CTA баннер** — Призыв к действию
10. **FAQ** — 8 вопросов и ответов
11. **Контакты** — Форма + контактная информация
12. **Footer** — Подвал с навигацией
